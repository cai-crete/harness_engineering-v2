# generated/ — 자동 생성 사실 레이어

> 이 폴더의 파일들은 사람이 직접 작성하지 않습니다.
> 코드베이스 또는 시스템 상태를 분석하여 자동으로 생성됩니다.
> Agent는 이 파일들을 **검증된 사실(verified fact)**로 취급합니다.

---

## 이 구분이 중요한 이유

사람이 작성한 문서는 코드가 변경될 때 함께 업데이트되지 않으면
실제 코드 상태와 점점 괴리됩니다.

`generated/` 파일들은 항상 코드에서 직접 추출되므로
문서와 코드 사이의 불일치가 발생하지 않습니다.

---

## 파일 작성 규칙

- 이 폴더에 파일을 수동으로 작성하지 않습니다
- 파일은 스크립트 또는 Agent가 코드베이스를 분석하여 생성합니다
- 생성된 파일 상단에 생성 일시와 생성 방법을 명시합니다

```
# [파일명]
# Generated: YYYY-MM-DD
# Source: [생성 스크립트 또는 생성 방법]
```
